# go-webapp-template
===========================

A template for Go Web Applications using: 
 * [Gorilla Toolkit](http://www.gorillatoolkit.org/)
 * [Negroni](https://github.com/urfave/negroni)
 * [nnmware / DevOOPS Bootstrap 3 Admin theme](https://github.com/nnmware/devoops)
